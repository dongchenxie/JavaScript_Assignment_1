Dates in JavaScript


Creating Date objects:

    
    let now=new Date()
    /*Generate a Date object in javascript
    with no parameter will get an Date object that
    shows how many milliseconds pasted by 1st Jan, 1970 0:00
    */
    let newYear=new Date(2020,0,1);
    /* A Date object can accpect parameters like Date(year,month,day,hour,minutes,second)*/

Caculate the duration between two days

    
    let now=new Date();
    let newYear=new Date(2020,0,1);
    /*copy from the previous code*/
    let differece=newYear-now;//gets the difference in milliseconds;
    let differenceInDays=differece/1000/60/60/24;//convert the difference to difference in days;
    console.log(differenceInDays.toFixed(2));//round up the days to two decimals and print it in the console;

Arrays and loops


Pushing elements into the end of an array

    
    let array=[1,2,3,4];
    console.log(array);//will print 1,2,3,4
    array.push(5);//push 5 to the last postion of the array
    console.log(array);//will print 1,2,3,4,5


Two ways to loop through an array

    
    /*regular for loop, it will print 1234*/

    for(let var=i;i<array.length;i++){
      console.log(array[i]);
